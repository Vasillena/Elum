## ⭐️  Introduction

ELUM is an upcoming nightclub concept designed to offer a different kind of nightlife experience. Since the physical space has not opened yet, there was no existing content — no events, no photography, no atmosphere to showcase. The challenge was clear: create anticipation and identity before the club physically exists. Instead of building a traditional website, I set out to design an experience — something immersive, cinematic, and emotionally charged. This is my first fully 3D web project.⁣
   <br/>
This project is built with Next.js, Tailwind, GSAP and Three.js.
   <br/>

View project at: https://elum.bg
<br/>

## 💡 Getting Started
Clone the repository:
```
git clone https://github.com/Vasillena/Elum
```
Run the following command to install all required packages and dependencies:
```
npm install
```
Run the following command to start the app in development mode:
```
npm run dev
```
Open http://localhost:3000 in your browser to view the application.
